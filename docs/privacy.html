Privacy Policy

Effective date: 2026-06-24  |  App: Blink Notifier

Summary

Blink Notifier is a wellness reminder app that helps you follow the 20-20-20 rule to reduce eye strain. It collects no personal data and makes no network connections.

Data we collect

None. Blink Notifier does not collect, store, transmit, or share any personal information, usage data, diagnostics, telemetry, or crash reports. Everything stays on your device.

Data stored locally

The app saves your settings (reminder interval, schedule, auto-launch preference) to a JSON file at %LOCALAPPDATA%\BlinkNotifier\settings.json. This file never leaves your device.

A rolling log file is written to %LOCALAPPDATA%\BlinkNotifier\logs\ for troubleshooting. Log files contain no personal information and are not transmitted anywhere. Files older than 7 days are deleted automatically.

Permissions

  • Notifications — to deliver timed eye-break reminders.
  • Run at startup (optional) — to start Blink Notifier with Windows if you enable auto-launch.
  • Foreground window detection — the app reads which window is currently active (using a standard Windows API, locally on your device) to detect fullscreen mode and suppress reminders while you are watching video or gaming. This information is never stored or transmitted anywhere.

No other system capabilities are requested.

Children

Blink Notifier is rated Everyone and does not knowingly collect information from children under 13.
